# المساهمة في مشروع نجم راش

شكراً لاهتمامك بالمساهمة في مشروع شركة نجم راش! نحن نقدر مساعدتك في تطوير وتحسين الموقع.

## 🚀 كيفية المساهمة

### 1. إعداد البيئة

```bash
# استنساخ المشروع
git clone <repository-url>
cd najm-next

# تثبيت التبعيات
npm install

# تشغيل الخادم
npm run dev
```

### 2. فهم هيكل المشروع

- `app/` - صفحات Next.js App Router
- `components/` - المكونات القابلة لإعادة الاستخدام
- `data/` - ملفات البيانات الثابتة
- `public/` - الصور والملفات الثابتة

### 3. إرشادات الكود

- استخدم ESLint و Prettier لتنسيق الكود
- اكتب تعليقات واضحة باللغة العربية
- استخدم TypeScript للمكونات الجديدة عند الإمكان
- اتبع اتفاقيات التسمية الموجودة

### 4. عمل Pull Request

1. Fork المشروع
2. أنشئ branch جديد: `git checkout -b feature/اسم-الميزة`
3. اجعل تغييراتك
4. تأكد من أن الاختبارات تمر: `npm run lint && npm run format:check`
5. Commit التغييرات: `git commit -m 'إضافة: وصف الميزة'`
6. Push للbranch: `git push origin feature/اسم-الميزة`
7. افتح Pull Request
