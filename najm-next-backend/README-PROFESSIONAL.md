# نجم راش للمقاولات - موقع إلكتروني احترافي

## نظرة عامة

موقع إلكتروني احترافي لشركة نجم راش للمقاولات، مصمم باستخدام Next.js 14 مع واجهة مستخدم عصرية ومتجاوبة.

## المميزات الرئيسية

### 🎨 تصميم احترافي

- تصميم عصري ومتجاوب مع جميع الأجهزة
- ألوان ذهبية مميزة تعكس الجودة والتميز
- تأثيرات بصرية متقدمة وانيميشن سلس
- واجهة مستخدم سهلة الاستخدام

### 📱 المكونات الرئيسية

#### الصفحة الرئيسية

- **Hero Section**: قسم بطول الشاشة مع صور متغيرة ومعلومات الشركة
- **About Section**: قصة الشركة وقيمها الأساسية
- **Stats Section**: إحصائيات الشركة مع عدادات متحركة
- **Services Section**: عرض تفاعلي للخدمات مع تفاصيل شاملة
- **Testimonials Section**: آراء العملاء مع تصميم احترافي
- **Projects Section**: عرض المشاريع مع فلترة حسب الفئة
- **CTA Section**: دعوة للعمل مع معلومات التواصل
- **Partners Section**: عرض الشركاء الاستراتيجيين

### Frontend

- **Next.js 14** - إطار العمل الرئيسي
- **React 18** - مكتبة واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS
- **React Icons** - مكتبة الأيقونات

## التثبيت والتشغيل

1. **استنساخ المشروع**

```bash
git clone <repository-url>
cd najm-next-backend
```

2. **تثبيت التبعيات**

```bash
npm install
```

3. **إعداد قاعدة البيانات**

```bash
npx prisma generate
npx prisma db push
```

4. **إعداد المتغيرات البيئية**

```bash
cp .env.example .env.local
# قم بتحرير .env.local بالقيم الصحيحة
```

5. **تشغيل الخادم**

```bash
npm run dev
```

6. **فتح المتصفح**

```
http://localhost:3000
```

## إعداد البريد الإلكتروني

لإعداد البريد الإلكتروني مع Hostinger:

1. أنشئ حساب Hostinger SMTP
2. احصل على بيانات SMTP
3. أضف البيانات إلى `.env.local`:

```
SMTP_HOST=your-smtp-host
SMTP_PORT=587
SMTP_USER=your-email@domain.com
SMTP_PASS=your-password
```
