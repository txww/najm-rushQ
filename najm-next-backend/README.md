# شركة نجم راش للمقاولات - موقع إلكتروني احترافي

موقع إلكتروني احترافي متطور لشركة نجم راش للمقاولات، مبني بأحدث التقنيات لضمان أعلى مستويات الأداء والأمان.

## 🌟 المميزات الرئيسية

### 🎨 تصميم متقدم
- **تصميم مريح للعين**: ألوان هادئة، تباعد مثالي، خطوط واضحة
- **متجاوب بالكامل**: يعمل بسلاسة على جميع الأجهزة
- **تأثيرات بصرية متطورة**: gradients، shadows، animations ناعمة
- **وضع مظلم**: دعم كامل للوضع المظلم والفاتح

### 🚀 أداء عالي
- **Next.js 16**: أحدث إصدار مع App Router
- **تحميل سريع**: تحسين الصور والخطوط تلقائياً
- **PWA جاهز**: قابل للتثبيت كتطبيق ويب تقدمي
- **SEO محسن**: محركات البحث صديقة

### 📊 لوحة تحكم شاملة
- **إحصائيات تفاعلية**: عرض بيانات حية للموقع
- **إدارة المحتوى**: مواد، منتجات، مشاريع، إعدادات
- **تقارير متقدمة**: تحليلات الأداء والزوار
- **واجهة سهلة الاستخدام**: تصميم حديث ومتجاوب

### 🔧 مميزات تقنية
- **قاعدة بيانات Prisma**: إدارة بيانات آمنة وفعالة
- **رفع الصور**: نظام رفع متقدم مع معالجة تلقائية
- **إرسال البريد**: تكامل مع Resend للبريد الإلكتروني
- **معالجة الأخطاء**: Error Boundaries شاملة
- **تحليلات Google**: تتبع الزوار والأداء

## 🐳 النشر السريع

### النشر المحلي مع Cloudflare Tunnel

1. **تشغيل النشر التلقائي:**
   ```bash
   ./deploy.bat
   ```

2. **إعداد Cloudflare Tunnel:**
   ```bash
   ./setup-tunnel.bat
   ```

3. **تشغيل الموقع:**
   ```bash
   # Terminal 1: الموقع
   npm run start:prod

   # Terminal 2: Cloudflare Tunnel
   cloudflared tunnel run najm-tunnel
   ```

### النشر بـ Docker

```bash
# بناء وتشغيل
docker-compose up -d

# أو يدوياً
docker build -t najm-next .
docker run -p 3000:3000 najm-next
```

## 📦 التثبيت والإعداد

### متطلبات النظام
- Node.js 18+
- npm أو yarn
- Git

### خطوات التثبيت

1. **استنساخ المشروع:**
```bash
git clone <repository-url>
cd najm-next-backend
```

2. **تثبيت التبعيات:**
```bash
npm install
```

3. **إعداد قاعدة البيانات:**
```bash
npx prisma generate
npx prisma db push
npm run seed
```

4. **إعداد متغيرات البيئة:**
انسخ `.env.example` إلى `.env.local` وأضف القيم المناسبة:

```env
# قاعدة البيانات
DATABASE_URL="file:./dev.db"

# البريد الإلكتروني
RESEND_API_KEY="your_resend_api_key"

# إعدادات الشركة
NEXT_PUBLIC_WHATSAPP_NUMBER="+963984966818"
NEXT_PUBLIC_COMPANY_NAME="شركة نجم راش للمقاولات"
NEXT_PUBLIC_PHONE="+966112345678"

# Google Analytics
NEXT_PUBLIC_GA_ID="G-XXXXXXXXXX"

# إعدادات التطوير
NODE_ENV="development"
```

5. **تشغيل المشروع:**
```bash
npm run dev
```

6. **افتح المتصفح:**
[http://localhost:3000](http://localhost:3000)
