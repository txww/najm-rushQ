# صفحة التواصل - Contact Page

## نظرة عامة
صفحة التواصل المتقدمة في موقع نجم راش للمقاولات، مصممة لتوفير تجربة مستخدم شاملة ومتقدمة.

## الميزات الرئيسية

### 1. تصميم التبويبات (Tabs)
- **نموذج الاتصال**: نموذج اتصال متقدم مع حقول متعددة
- **معلومات الشركة**: عرض شامل لمعلومات الشركة وخدماتها
- **الأسئلة الشائعة**: إجابات على الأسئلة الأكثر شيوعاً

### 2. نموذج الاتصال المتقدم
- حقول أساسية: الاسم، البريد الإلكتروني، رقم الهاتف
- حقول إضافية: نوع المشروع، درجة الاستعجال، الموضوع، التفاصيل
- التحقق من صحة البيانات
- إرسال البريد الإلكتروني عبر API
- رسالة تأكيد الإرسال

### 3. طرق التواصل السريع
- WhatsApp: رابط مباشر للمحادثة
- الهاتف: رابط مباشر للاتصال
- البريد الإلكتروني: رابط لإرسال بريد
- الموقع: رابط لخريطة الموقع

### 4. ساعات العمل
- عرض ساعات العمل لكل أيام الأسبوع
- ملاحظة خاصة بتوفر WhatsApp 24/7

### 5. إجراءات سريعة
- روابط سريعة للصفحات الرئيسية:
  - خدماتنا
  - مشاريعنا
  - WhatsApp
  - من نحن

### 6. معلومات الشركة
- نبذة عن الشركة
- الخدمات الرئيسية
- أسباب اختيار الشركة
- معلومات التواصل الكاملة

### 7. الأسئلة الشائعة
- أسئلة حول وقت الرد
- الزيارات الميدانية
- عروض الأسعار
- طرق الدفع
- الضمان
- متابعة المشاريع

## التقنيات المستخدمة

### Frontend
- **Next.js 14**: App Router
- **React 18**: Hooks (useState, useEffect, useRef)
- **Tailwind CSS**: تصميم متجاوب
- **React Icons**: أيقونات FontAwesome

### Backend Integration
- **API Routes**: `/api/send-email` لإرسال البريد
- **Environment Variables**: إعدادات الاتصال
- **Form Validation**: تحقق من صحة البيانات

### UI/UX Features
- **Responsive Design**: تصميم متجاوب لجميع الأجهزة
- **Smooth Animations**: انتقالات سلسة
- **Loading States**: مؤشرات التحميل
- **Error Handling**: معالجة الأخطاء
- **Scroll to Top**: زر العودة لأعلى
